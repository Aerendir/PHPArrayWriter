A class to write and read arrays.

## Available methods

- `getValue()`
- `getValueByPartialKey()`
- `isNode()`
- `isReadable()`
- `isRoot()`
- `isWritable()`
- `keyExistsNested()`
- `add()`
- `cp()`
- `cpSafe()`
- `edit()`
- `merge()`
- `mv()`
- `mvSafe()`
- `mvUp()`
- `rm()`
- `wrap()`
- `pathize()`
- `unpathize()`
- `forceArray()`

For details about each method, please, read the comments in the code: they are really simple to be read, trust me!
